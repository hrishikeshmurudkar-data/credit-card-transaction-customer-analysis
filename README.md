</p>📌 Project Overview

This project analyzes credit card transactions and customer demographics to uncover spending patterns, revenue trends, and customer behavior.

An interactive Power BI dashboard is developed to visualize key business metrics and help stakeholders make data-driven decisions.

---

🎯 Business Problem

Financial institutions process thousands of credit card transactions daily. Understanding customer spending patterns helps organizations:

• Identify high-value customers
• Monitor revenue performance
• Understand spending categories
• Improve financial decision-making

This project provides insights using transaction and customer data.

---

📂 Dataset

The dataset contains two main tables:

1️⃣ Credit Card Transaction Data

Includes information such as:

- Transaction amount
- Transaction category
- Card type
- Transaction date

2️⃣ Customer Data

Contains:

- Customer age
- Gender
- Income group
- State
