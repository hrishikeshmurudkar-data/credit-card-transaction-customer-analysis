</p>📊 Project Overview

This project analyzes credit card transactions and customer demographics to uncover spending patterns, revenue trends, and customer behavior.

An interactive Power BI dashboard was developed to visualize key business metrics and support data-driven decision making.

---

🎯 Business Problem

Financial institutions process thousands of credit card transactions daily. Understanding customer spending patterns and transaction behavior helps businesses:

• Identify high-value customers
• Monitor revenue trends
• Understand spending categories
• Improve financial decision making

---

📂 Dataset

The dataset consists of two primary tables.

Credit Card Transaction Data

- Transaction Amount
- Transaction Category
- Card Category
- Transaction Date

Customer Data

- Age
- Gender
- Income Group
- State

---

🛠 Tools Used

- Power BI – Dashboard Development
- SQL – Data Querying
- Excel – Data Preparation
- DAX – Data Modeling

💡 Key Insights

• Certain credit card categories generate higher revenue
• Spending behavior varies across different age groups
• Some regions contribute significantly to overall transactions
• Entertainment and bill payments dominate spending categories
